## Tools and services involved
### LLM services
The LLM service in this course is powered by ModelScope. It is openai-compatible, which means you can use the same API to interact with it.

### WebSearch Engine
We use Firecrawl as the web search engine service.

### ERP website
We use jshERP as the ERP system for this course.
https://github.com/jishenghua/jshERP

### IDE
We recommend using PyCharm as the IDE for this course.
